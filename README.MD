# Estruturas Avançadas de Árvores

Trabalho avaliativo da disciplina de **Estruturas de Dados** — UniCesumar.

O objetivo foi pesquisar, documentar e comparar estruturas avançadas de árvores, relacionando os conceitos teóricos com aplicações reais da computação.

---

## Conteúdo

O arquivo `trabalho-arvores.md` está organizado em quatro partes:

**Parte 1 – Tipos de Árvores**
Conceito, características, vantagens, desvantagens e exemplos ilustrados das seguintes estruturas:

- AVL
- Rubro-Negra
- N-ária

**Parte 2 – Operações em Árvores**
Explicação e exemplos passo a passo das operações de rebalanceamento:

- Rotação Simples à Direita (caso LL)
- Rotação Simples à Esquerda (caso RR)
- Rotação Dupla Esquerda-Direita (LR)
- Rotação Dupla Direita-Esquerda (RL)
- Inversão (Espelhamento)

**Parte 3 – Aplicação Prática**
Análise de uso real: indexação em bancos de dados relacionais com **Árvore B+**, justificando a escolha com base em desempenho, organização dos dados e operações do sistema.

**Parte 4 – Comparação entre Estruturas**
Tabela comparativa entre BST, AVL, Rubro-Negra e N-ária considerando número de filhos, balanceamento, complexidade de busca e inserção, vantagens, desvantagens e exemplos de aplicação.
